# 🤖 Prompts Reutilizáveis para Estudos com IA

Este arquivo reúne prompts desenvolvidos durante o projeto para auxiliar estudos utilizando Inteligência Artificial como ferramenta de aprendizagem.

---

# 1. Explicação de conceitos

## Objetivo:
Compreender um conceito novo utilizando linguagem simples.

## Prompt:

Explique [CONCEITO] para uma pessoa iniciante.

Utilize linguagem simples, exemplos do cotidiano e organize a resposta em tópicos.

---

# 2. Explicação como professor

## Objetivo:
Aprofundar o entendimento de um assunto.

## Prompt:

Explique [TEMA] como um professor ensinando um aluno iniciante.

Apresente:

- definição;
- principais características;
- exemplos práticos;
- aplicações no cotidiano;
- resumo final.

---

# 3. Comparação de conceitos

## Objetivo:
Entender diferenças entre dois assuntos.

## Prompt:

Compare [CONCEITO A] e [CONCEITO B].

Mostre:

- principais diferenças;
- características;
- vantagens e desvantagens;
- exemplos práticos;
- tabela comparativa.

---

# 4. Criação de resumo

## Objetivo:
Transformar um conteúdo extenso em material de revisão.

## Prompt:

Crie um resumo estruturado sobre [TEMA].

Inclua:

- conceitos principais;
- pontos importantes;
- exemplos;
- aplicações práticas;
- conclusão.

---

# 5. Criação de glossário

## Objetivo:
Criar uma referência rápida dos principais termos.

## Prompt:

Crie um glossário sobre [TEMA].

Liste os principais termos e explique cada conceito utilizando linguagem simples e objetiva.

---

# 6. Explicação passo a passo

## Objetivo:
Aprender cálculos ou processos.

## Prompt:

Explique [PROCESSO/CÁLCULO] passo a passo.

Mostre:

- cada etapa;
- explicação dos elementos utilizados;
- exemplo resolvido;
- resumo final.

---

# 7. Transformar conteúdo em material de estudo

## Objetivo:
Criar anotações organizadas.

## Prompt:

Transforme este conteúdo em anotações de estudo.

Utilize:

- títulos;
- tópicos;
- definições curtas;
- exemplos importantes.

Não adicione informações que não estejam no conteúdo original.

---

# 8. Revisão antes de uma prova

## Objetivo:
Revisar conhecimento adquirido.

## Prompt:

Faça uma revisão sobre [TEMA].

Inclua:

- conceitos que preciso lembrar;
- erros comuns;
- perguntas para testar meu conhecimento;
- resumo final.
